Disciplina: Segurança de Sistemas

Este projeto consiste na implementação de um sistema de criptografia assimétrica em Python, desenvolvido como atividade prática da disciplina de Segurança de Sistemas.

O código foi inspirado no conteúdo educacional apresentado por Alestan Alves (Ale) em seu vídeo no YouTube. No entanto, para esta atividade, foi utilizada a biblioteca cryptography.
